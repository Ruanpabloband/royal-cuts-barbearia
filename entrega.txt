4. COMO FUNCIONA O AGENDAMENTO
-----------------------------------------

O cliente entra no site, escolhe o servico, data e horario,
preenche nome e telefone, e clica "Agendar via WhatsApp".

O sistema verifica se o horario esta disponivel em tempo real.
Se estiver livre, reserva o horario e redireciona para o WhatsApp
com todos os dados do agendamento formatados.

IMPORTANTE: O barbeiro deve confirmar o agendamento pelo WhatsApp.
O horario fica reservado por 48 horas no sistema.

7. SERVICOS COM CONTROLE DE HORARIOS
-----------------------------------------

Os horarios sao controlados em tempo real pelo Vercel KV (banco Redis).
Quando um cliente agenda um horario, ele fica reservado no sistema.
Outros clientes veem apenas os horarios disponiveis.

Para acessar o painel do Vercel KV:
1. Acesse vercel.com/dashboard
2. Selecione o projeto
3. Va em Storage > KV
4. La voce pode ver os agendamentos registrados

Free tier: 30.000 requests/dia (mais que suficiente)
